# 🔗 Linked List Visualizer (Django)

A web-based **Data Structures Visualizer** built with Python and Django that demonstrates operations on a **Singly Linked List** such as insertion and deletion with a graphical representation.

---

## 📌 Features

- Insert node at beginning
- Insert node at end
- Insert node in middle
- Delete node
- Visual representation of linked list nodes
- Interactive UI using Bootstrap

---

## 🛠 Tech Stack

**Backend**
- Python
- Django

**Frontend**
- HTML
- CSS
- Bootstrap

**Concepts Used**
- Data Structures
- Singly Linked List
- MVC Architecture (Django)
